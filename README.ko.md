# SpecVital

[English](README.md)

GitHub 레포지토리의 테스트 코드를 분석하고 시각화합니다.

**[specvital.com에서 바로 사용하기](https://specvital.com)**

## 주요 기능

- **테스트 분석** — GitHub 레포 URL만 입력하면 테스트 스위트, 케이스, 상태를 한눈에 파악
- **SpecView** — AI가 테스트 코드에서 행동 명세를 자동 생성
- **20+ 프레임워크** — 언어와 테스트 러너에 관계없이 바로 사용 가능

## 지원 프레임워크

| 언어          | 프레임워크                               |
| ------------- | ---------------------------------------- |
| JavaScript/TS | Jest, Vitest, Mocha, Playwright, Cypress |
| Go            | testing, GTest                           |
| Python        | pytest, unittest                         |
| Java/Kotlin   | JUnit 4, JUnit 5, TestNG, Kotest         |
| Ruby          | RSpec, Minitest                          |
| C#            | xUnit, NUnit, MSTest                     |
| PHP           | PHPUnit                                  |
| Swift         | XCTest, Swift Testing                    |
| Rust          | Cargo Test                               |
