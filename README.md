# SpecVital

[한국어](README.ko.md)

Analyze and visualize test suites from any GitHub repository.

**[Try it now at specvital.com](https://specvital.com)**

## Features

- **Test Analysis** — Paste a GitHub repo URL, get a full breakdown of test suites, cases, and status
- **SpecView** — AI-generated behavior specs from your test code
- **20+ Frameworks** — Works across languages and test runners out of the box

## Supported Frameworks

| Language      | Frameworks                               |
| ------------- | ---------------------------------------- |
| JavaScript/TS | Jest, Vitest, Mocha, Playwright, Cypress |
| Go            | testing, GTest                           |
| Python        | pytest, unittest                         |
| Java/Kotlin   | JUnit 4, JUnit 5, TestNG, Kotest         |
| Ruby          | RSpec, Minitest                          |
| C#            | xUnit, NUnit, MSTest                     |
| PHP           | PHPUnit                                  |
| Swift         | XCTest, Swift Testing                    |
| Rust          | Cargo Test                               |
