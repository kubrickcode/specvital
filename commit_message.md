# Commit Message

## Korean Version

```
refactor(ui): 필터 버튼 높이를 h-9로 통일

필터 버튼(size="sm", h-8)과 검색 Input(h-9) 간 높이 불일치로 필터 바의 시각적 안정감이 떨어짐.

size="default"(h-9)로 변경하여 검색 Input과 높이를 맞추고, 모바일용 min-h-[44px] 반응형 스타일 제거 (h-9=36px가 WCAG AA 터치 타겟 기준 24px 충족).
```

## English Version

```
feat(ui): unify filter button heights to h-9

Filter buttons (size="sm", h-8) misaligned with search Input (h-9), causing visual inconsistency in filter bar.

Changed to size="default" (h-9) to match Input height and removed mobile min-h-[44px] responsive styles (h-9=36px meets WCAG AA touch target 24px).
```
